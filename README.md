# AI Resume Parser

An NLP-based web application that extracts important information from resumes automatically.

## Features
- Resume Upload
- Skill Extraction
- Education Detection
- Experience Parsing
- PDF Resume Processing

## Tech Stack
- Python
- Flask
- NLP
- spaCy
- SQLite

## Project Structure

```bash
resume-parser/
│
├── app.py
├── database.db
├── resumes/
├── templates/
└── README.md
```

## How to Run

1. Clone repository

```bash
git clone https://github.com/divya16-valli/resume-parser.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run application

```bash
python app.py
```
